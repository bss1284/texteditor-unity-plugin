## Text Editor for Unity
<img src="Docs/capture1.png?raw=true" alt="Example" width="512px" height="512px"/>

## FEATURES 
- Editing
- Save (Ctrl+S)
- Revert (Ctrl+R)
- Undo Save

## HOW TO
1. Go to 'Tools/BSS/Editor/Text Editor'
2. Open Text Editor Window.
3. Select Text Asset.
4. Editing and Save.
