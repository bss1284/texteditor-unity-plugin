Text Editor for Unity

## FEATURES 
- Editing
- Save (Ctrl+S)
- Revert (Ctrl+R)
- Undo Save

## HOW TO
1. Go to 'Tools/BSS/Editor/Text Editor'
2. Open Text Editor Window.
3. Select Text Asset.
4. Editing and Save.
